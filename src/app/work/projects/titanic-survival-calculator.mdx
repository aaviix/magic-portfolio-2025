## Overview

The Titanic Survival Calculator is an end-to-end web application that leverages machine learning to predict the survival chances of Titanic passengers. It features a FastAPI backend, a VueJS frontend, and utilizes Docker for containerization and CI/CD for continuous integration.

## Key Features

- **Predictive Modeling:** Uses historical data to predict survival outcomes.
- **API-Driven:** FastAPI powers the backend, providing robust RESTful endpoints.
- **Interactive UI:** VueJS creates an engaging and user-friendly interface.
- **DevOps Ready:** Dockerized services with automated CI/CD pipelines ensure reliable deployments.

## Technologies Used

- **Python & FastAPI**
- **VueJS**
- **Docker**
- **GitLab CI**, **pytest**, and **Cypress/Playwright**

## Outcome

The Titanic Survival Calculator illustrates the integration of machine learning with modern web development and DevOps practices, resulting in a scalable and efficient prediction tool.
