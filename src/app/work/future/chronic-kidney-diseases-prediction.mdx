## Overview

This project is an interactive application designed to predict the likelihood of chronic kidney disease using a logistic regression model. A PyQt6-based graphical user interface makes the tool accessible and easy to use for early detection.

## Key Features

- **Predictive Analytics:** Implements logistic regression for disease prediction.
- **User-Friendly GUI:** Developed with PyQt6 for an intuitive user experience.
- **Early Warning:** Provides actionable insights for proactive health management.

## Technologies Used

- **Python** for model development.
- **Logistic Regression** for prediction.
- **PyQt6** for the GUI.

## Outcome

The tool effectively demonstrates how predictive analytics can be applied in healthcare, offering early warnings that can aid in the proactive management of kidney health.
